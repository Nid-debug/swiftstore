# 🛒 SwiftStore - E-Commerce Frontend

A modern, responsive e-commerce frontend built using React and Redux Toolkit.
This project focuses on real-world UI/UX and state management patterns.

---

## 🚀 Features

- 🛍️ Product listing with clean UI
- 🔍 Search functionality (Navbar integrated)
- 📂 Category-based filtering
- ❤️ Wishlist system
- 🛒 Cart (Bag) with quantity management
- 💰 Dynamic pricing & order summary
- 🔗 URL-based filtering (shareable links)
- 📱 Fully responsive design

---

## 🧠 Key Concepts Learned

- State management using Redux Toolkit
- URL query params handling with React Router
- Component reusability and structure
- UI/UX improvements with Tailwind CSS
- Conditional rendering & performance optimization

---

## 🛠️ Tech Stack

- React.js
- Redux Toolkit
- React Router DOM
- Tailwind CSS
- FontAwesome + Lucide Icons

---

## 📦 Installation

```bash
git clone https://github.com/Nid-debug/swiftstore.git
cd swiftstore
npm install
npm run dev
```
